# Switch to English

## Overview

Switch the current session's dialog language to English.

## Steps

1. **Confirm switch**
   - Confirm to the user: dialog language has been switched to English

2. **Subsequent behavior**
   - From this point on, all dialog, explanations, and questions use English
   - Code, code comments, and tool call descriptions remain in English as always

## Rules

- Takes effect immediately
- Only affects dialog language, code and comments always stay in English
- To switch to Chinese, use `/chinese-talk`
