# Switch Dialog to Chinese

## Overview

Switch the current session's dialog language to Chinese. All code, comments, and tool call descriptions remain in English.

## Steps

1. **Confirm switch**
   - Confirm to the user in Chinese: dialog language has been switched to Chinese

2. **Subsequent behavior**
   - From this point on, all dialog, explanations, and questions use Chinese
   - Code, code comments, and tool call descriptions remain in English, unaffected

## Rules

- Takes effect immediately
- Only affects dialog language, code and comments always stay in English
- To switch back to English, use `/english-talk`
